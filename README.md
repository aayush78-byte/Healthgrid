# HealthGrid - AI-Powered Healthcare Ecosystem

HealthGrid is a modern healthcare platform designed to bridge the gap between citizens and hospitals. It leverages AI to provide instant health insights, interactive mapping for hospital discovery, and streamlined administrative tools for medical facilities.

## Key Features

### For Citizens
- **AI Health Assistant**: Get instant insights and wellness advice based on Ayush and modern medical standards.
- **Find Hospitals**: Interactive map identifying nearby hospitals with real-time bed availability.
- **Health Records**: Manage and access your medical history securely.
- **Emergency Alerts**: Stay informed about local health risks and outbreaks.

### For Hospitals
- **Bed Management**: Real-time tracking and management of ICU, General, and Emergency beds.
- **Patient Records**: Streamlined access to patient data and diagnostic history.
- **Disease Analytics**: Comprehensive visualizations of health trends and disease outbreaks.
- **Dual Coding Engine**: Intelligent mapping between traditional Ayush codes and global ICD-11 standards.

## Tech Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Maps**: Leaflet & React Leaflet
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation
1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Development
- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run test`: Run unit tests with Vitest
- `npm run lint`: Run ESLint for code quality
